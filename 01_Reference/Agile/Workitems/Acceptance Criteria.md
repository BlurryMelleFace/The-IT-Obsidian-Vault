# Acceptance Criteria Prompt

> [!TIP]
> Use this prompt to generate rigorous test cases (Gherkin style) for your User Stories.

```text
**Please create test cases that I can include in the acceptance criteria of the PBI.**
The test cases must follow this format:
- **Test Case Number:**
- **Given** …
- **When** …
- **Then** …
Each test case should be described briefly and as concisely as possible.
The acceptance criteria and their corresponding test cases refer to the following PBI: _[Infos appended]_
```
